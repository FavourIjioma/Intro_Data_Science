## Tools:

### *Python*

**Python** is one of the most popular programming languages used to give specific instructions to a computer. It allows developers to communicate with computers by writing codes that tells the computer what tasks to perform. It is a powerful and beginner-friendly tool used for automation and scripting, web development, data science and analysis, to mention a few.

### *Marimo notebooks*

We can call this the "smart notebook for coding." It's a tool that allows users to write and test python code and seeing the results immediately, and any changes made to the code automatically updates the results.

### *Git Version Control*

To the best of my understanding, this is simply a collaborative tool that allows multiple developers to work on a product without affecting each other's work. This tool tracks every changes, and allows users to revert to previous and original versions.

### *Plotly Visualization*

**Plotly** is a powerful visualization library used in creating high quality charts, with features that are interactive, beautiful, versatile, and web-ready.

### *Polars Data Wrangling*

**Polars** is simply a Python library/tool for working with tables of data. It is used for cleaning, organizing, transforming, and analyzing data.

### *Visual Studio (VS) Code*

VS Code is a beginner and expert-friendly Microsoft tool just like the MS Word, but **specifically for writing codes.** It catches errors as codes are written, and carries out other powerful functions.

### *UV Package Management*

This is an extremely fast tool used in Python to download and organize the extra tools a program needs to run.

## Data Science Guide

This is simply the practice of turning  raw data into actionable information that help organizations and sectors make better decisions. **Data science** involves;

1. Gathering data from different sources and preparing it for analysis
2. Exploring the data to find patterns
3. Build preditive models
4. Communicate findings

This practice provides **powerful data-driven decisions** - which has impacted the world greatly, and to succeed in it, the following skills are needed;

- Critical Thinking
- Programming
- Statistics and Mathematics
- Communication
- Data Visualization
- Domain Knowledge
- Machine Learning
